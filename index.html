클라우드 동기화

JSONBin.io 가입 → Bin 생성(Private 권장) → API Key 입력. PC·폰에 같은 값을 넣으면 자동으로 양방향 동기화됩니다. 비워두면 이 기기에만 저장됩니다.

보안 팁: 계정 전체 권한인 마스터 키 대신, 이 Bin에만 권한을 준 Access Key 사용을 권합니다. 키가 노출돼도 이 Bin만 영향받습니다.
