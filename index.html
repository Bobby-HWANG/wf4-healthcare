로컬 모드

클라우드 동기화 (JSONBin.io)

JSONBin.io 가입 → Bin 생성(Private 권장) → API Key 입력. PC·폰에 같은 값을 넣으면 자동으로 양방향 동기화됩니다. 비워두면 이 기기에만 저장(로컬 모드)됩니다.

보안 팁: 가능하면 계정 전체 권한인 마스터 키 대신, 이 Bin에만 권한을 준 Access Key를 발급해 쓰세요. 키가 노출돼도 이 Bin만 영향을 받습니다.

로컬 모드일 때도 데이터는 이 브라우저에 안전하게 보관됩니다.
